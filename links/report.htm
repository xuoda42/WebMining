Отчет об информационном поиске

Цель исследования: Поиск и систематизация материалов по теме выпускной работы: «Интеллектуальный планировщик задач».

В рамках работы выполнялся тематический информационный поиск, чтобы собрать актуальные материалы для раздела ссылок и библиотеки источников персонального сайта магистра. Основная тема — разработка интеллектуального планировщика задач в формате веб‑приложения.

При анализе рассматривались подходы к управлению задачами (в том числе Kanban), проектирование REST API на FastAPI, вопросы аутентификации (JWT), моделирование данных (PostgreSQL + SQLAlchemy), а также формализация жизненного цикла задач через конечные автоматы и правила переходов между состояниями.

Поиск информации осуществлялся с использованием следующих поисковых систем:

  • Google
  • Яндекс
  • Bing
  • Rambler

Для каждой поисковой системы формировались запросы на русском и английском языках, отражающие ключевые аспекты темы.

При формировании поисковых запросов учитывались особенности работы поисковых систем:

  • использование кавычек для точных фраз;
  • уточнение языка поиска для англоязычных запросов (Advanced Search → language: English);
  • избегание дублирующихся результатов, особенно при использовании метапоисковых систем.

По суммарному количеству найденных результатов наиболее продуктивной оказалась поисковая система Google: она стабильно давала максимальные значения как по русскоязычным, так и по англоязычным запросам. Bing также показал высокий охват, особенно для англоязычных формулировок.

Яндекс оказался полезен прежде всего для русскоязычных запросов, обеспечивая более «локальную» выдачу. Rambler во всех запросах показал минимальные значения, поэтому его вклад в подбор релевантных материалов оказался наименьшим.

В целом, собранные результаты позволили определить наиболее эффективные поисковые системы и уточнить набор ключевых запросов для дальнейшего поиска публикаций, документации и практических примеров по разработке интеллектуального планировщика задач.
