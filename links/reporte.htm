Information Search Report

Research objective: Searching for and systematizing materials on the master’s thesis topic: «Intelligent task planner».

As part of the work, a thematic information search was conducted to collect up-to-date materials for the links section and the source library of the master’s personal website. The main topic is the development of an intelligent task planner as a web application.

The analysis considered task management approaches (including Kanban), REST API design with FastAPI, authentication issues (JWT), data modeling (PostgreSQL + SQLAlchemy), and formalizing the task life cycle using finite-state machines and rules for transitions between states.

The information search was carried out using the following search engines:

  • Google
  • Yandex
  • Bing
  • Rambler

For each search engine, queries were formulated in Russian and English, reflecting the key aspects of the topic.

  • using quotation marks for exact phrases;
  • specifying the search language for English-language queries (Advanced Search → language: English);
  • avoiding duplicate results, especially when using metasearch engines.

By the total number of results found, Google turned out to be the most productive search engine: it consistently produced the highest counts for both Russian-language and English-language queries. Bing also showed broad coverage, especially for English wording.

Yandex was primarily useful for Russian-language queries, providing more “local” results. Rambler showed the lowest counts across all queries, so its contribution to selecting relevant materials was the smallest.

Overall, the collected results made it possible to identify the most effective search engines and refine the set of key queries for further searching of publications, documentation, and practical examples on developing an intelligent task planner.
